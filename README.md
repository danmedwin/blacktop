# Blacktop

A line-art racing game — black lines on paper white, the way racing games
looked on the classic Macintosh. An homage to a game my dad made years ago.

**Play it:** https://techrabbi.org/blacktop

- Three tracks: Figure Eight, Grand Circuit, Hairpin Ridge
- Cockpit view (windshield, gauges, steering wheel) or hood view — toggle with **V**
- Dodge the potholes; going off-road costs you dearly
- Fully manual steering — brake before the hairpins
- Three timed laps; best times saved per track

## Controls

| Key | Action |
| --- | --- |
| ↑ / ↓ | gas / brake |
| ← / → | steer |
| V | cockpit / hood view |
| M | engine sound on/off |
| P or Esc | pause |

Single self-contained HTML file, no dependencies. Keyboard required.
